🛒 E-Commerce Web Application

A full-stack E-Commerce platform built using Spring Boot + React, designed to provide seamless online shopping with secure authentication and scalable backend architecture.

📌 Project Overview

This project is a modern E-Commerce web application that allows users to:

Browse products by categories

Search products

Add items to cart

Manage orders

Securely authenticate using JWT

Perform CRUD operations via REST APIs

The application demonstrates strong knowledge of Java Full Stack Development, RESTful services, database design, and frontend integration.

🏗️ System Architecture

Backend: Spring Boot (REST APIs)

Frontend: React

Authentication: JWT

Database: MySQL / PostgreSQL

Microservices-ready structure
